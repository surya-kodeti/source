    About¶

    This is the documentation for the Objective-C/Swift OpenCV wrapper

    To get started: add the OpenCV framework to your project and add the following to your source

    Objective-C¶

    #import <OpenCV/OpenCV.h>

    Swift¶

    import OpenCV
